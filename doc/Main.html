type SearchCommand = String

Type of a SearchCommand

SearchCommand means:

  • "search hello t hs" (finding all files that the name include the word "hello" and the type is ".hs")
  • "search hello t lhs ms 40" (finding all files that the name include the word "hello", the type is ".lhs" and the size is lesser or equal than 40bytes)
  • etc...
